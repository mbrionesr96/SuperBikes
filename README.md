# SuperBikes

This GitHub repository is the space where Maria, Sayatan and Riccardo are storing the shared codes for the MSc Data Science for Management. 

The name of repository comes from the initial project SuperBikes of the Data Management course. This course project aimed to create a relational database usin PostgreSQL language for a bike rental system in Milan. We provided DDL, DML scripts for eachs table and two scripts with queries. 

The Pokemon folder is used for the course project on Data Visualization with R.
